Paradise Nursery is a React-based plant shopping web application that allows users to browse plants, add them to a shopping cart, update quantities, and manage their selections easily.

## Features
- Display a list of plants with images and prices
- Add plants to the shopping cart
- Increase or decrease item quantity
- Remove items from the cart
- View total items and total price
- Responsive and user-friendly UI

## Technologies Used
- React.js
- Redux Toolkit
- JavaScript (ES6)
- CSS
